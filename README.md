# RPG Terminal game com Swift - apple dev academy
Jogo de RPG no terminal usando Swift. Challenge do curso do processo seletivo Apple Dev Academy durante os dias 10/11 a 14/11

## Proposta
O jogo consiste em um mundo de fantasia em que um Dragao Laranja esta assombrando as pessoas do reino de Mon Ke Po e voce é o Heroi que irá salvar a todos!
Voce tem a opcao de batalhar para ficar forte e enfrentar o Dragao Laranja, enfrentando diversas criaturas magicas no caminho. Ao juntar 5 pontos, voce pode entrar na Caverna do Dragao e enfrenta-lo em uma batalha emocionante. 

Aventure-se no reino de Mon Ke Po e seja o Heroi!

## Como rodar
Baixe a pasta [Igor Teixeira Carrasco](https://github.com/BGNS42/RPG-game-com-Swift---apple-dev-academy/tree/main/Igor%20Teixeira%20Carrasco%20), abra o arquivo [main.swift](https://github.com/BGNS42/RPG-game-com-Swift---apple-dev-academy/blob/main/Igor%20Teixeira%20Carrasco%20/helloWorldApp/main.swift), que está dentro da pasta HelloWorldApp.

Abra o arquivo no Xcode ou acesse [Programiz - editor de codigo swift online](https://www.programiz.com/swift/online-compiler/), cole o código presente no `main.swift` e execute o código.

[Link direto para o editor online com o código nele](https://www.programiz.com/online-compiler/3Z2x2fDX6c4GA)

O jogo roda todo em terminal, com interacoes e sistema de batalha.
